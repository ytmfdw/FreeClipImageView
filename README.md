# FreeClipImageView
自定义裁剪图片控件
控件：MyImageView.java
功能：显示图片，裁剪指定区域内的图片，可进行手势移动，缩放指定区域，双击回调getBitmap方法，得到裁剪图片，请注意：Bitmap最好不要通过Intent传递
见博文：http://blog.csdn.net/ytmfdw/article/details/72819077
